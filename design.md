# GitHub Profile Page - Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a1d29) - Professional, trustworthy, tech-focused
- **Secondary**: Electric Blue (#3b82f6) - Modern, innovative, Azure-inspired
- **Accent**: Cyan (#06b6d4) - Data visualization, tech sophistication
- **Neutral**: Slate Gray (#64748b) - Supporting text, subtle elements
- **Background**: Dark gradient from navy to charcoal (#1a1d29 to #0f172a)

### Typography
- **Display Font**: "Inter" - Clean, modern sans-serif for headings and navigation
- **Body Font**: "JetBrains Mono" - Monospace for code snippets and technical content
- **Accent Font**: "Space Grotesk" - Contemporary geometric for special elements

### Visual Language
- **Minimalist Tech Aesthetic**: Clean lines, generous whitespace, focused content hierarchy
- **Data-Driven Visualization**: Interactive charts, progress indicators, skill matrices
- **Professional Sophistication**: Subtle animations, refined color transitions
- **GitHub Integration**: Octocat icons, repository-style cards, code block aesthetics

## Visual Effects & Styling

### Background Effects
- **Animated Gradient Flow**: Subtle aurora-like gradient animation using CSS and JavaScript
- **Particle Network**: Lightweight particle system representing data connections
- **Shader Effects**: Subtle noise and distortion for depth (using shader-park)

### Interactive Elements
- **Hover Transformations**: 3D tilt effects on cards, smooth scale transitions
- **Color Morphing**: Elements shift between blue and cyan on interaction
- **Glow Effects**: Soft neon-style glows on interactive elements
- **Smooth Reveals**: Content fades and slides in as user scrolls

### Animation Library Usage
- **Anime.js**: Primary animation engine for smooth micro-interactions
- **ECharts.js**: Interactive data visualizations for skills and project metrics
- **Splitting.js**: Text effects for headings and emphasis
- **Typed.js**: Typewriter effect for dynamic text content

### Header/Hero Effects
- **Gradient Text Animation**: Cycling colors on main heading
- **Floating Elements**: Subtle floating animation on profile image and icons
- **Backdrop Blur**: Glass-morphism effect on navigation and cards
- **Depth Layering**: Multiple z-index layers for visual hierarchy

### Content Styling
- **Card Design**: Rounded corners, subtle shadows, hover elevation
- **Code Blocks**: Dark theme with syntax highlighting
- **Progress Indicators**: Animated progress bars and skill meters
- **Icon Integration**: Custom SVG icons for technologies and skills

### Responsive Design
- **Mobile-First**: Optimized for all screen sizes
- **Touch Interactions**: Enhanced hover states for mobile devices
- **Flexible Layouts**: CSS Grid and Flexbox for adaptive content
- **Performance Optimized**: Lightweight animations and efficient rendering

## Component Aesthetics

### Navigation Bar
- Fixed position with backdrop blur
- Smooth transitions between sections
- Active state indicators with animated underlines

### Hero Section
- Large profile image with subtle border animation
- Animated text with gradient effects
- Call-to-action buttons with hover transformations

### Skills Section
- Interactive radar chart with smooth animations
- Color-coded skill categories
- Hover states revealing detailed information

### Project Cards
- Grid layout with consistent spacing
- Image overlays with gradient masks
- Animated reveals on scroll

### Blog Section
- Card-based layout with featured images
- Reading time and metadata
- Smooth transitions between articles
